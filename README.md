## Build

Install golang. 

Clone this repository:

```bash
git clone https://github.com/nyangkosense/gfetch
cd gfetch
```

Then build and install:

```bash
make && make install
```

There are 2 parameters:

- Display more info: `--full`
- Select logo: `--logo debian`

Example:

```bash
gfetch --full --logo debian
```
